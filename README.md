# Unity_UniversalMonoAutoBindingSystem
Unity
使用方法下载后直接放进自己Editor里即可

该工具可以针对所有继承MonoBehaviour的脚本使用   当然任意组件   

自动添加绑定可以收集子集所有出现过的组件选择性添加     也可以手动添加

命名空间管理可以方便的更改添加删除命名空间
